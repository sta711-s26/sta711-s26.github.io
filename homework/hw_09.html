t-tests

Here I want to revisit the topic of t-tests from the previous homework.

Why a t-test?

The use of the \(t\) distribution for testing hypotheses about means can be traced back to statistician William Sealy Gosset’s work while a researcher at the Guinness Brewery in Dublin. Unlike some other applications of statistics at the time (e.g. Pearson’s biometrics work, which could involve observations on hundreds of individuals), Gosset was particularly interested in inference with small sample sizes, for which the asymptotic results that we use so much don’t apply!

To make this concrete, suppose that \(X_1,...,X_n \overset{iid}{\sim} N(\mu, \sigma^2)\). We wish to test the hypotheses \(H_0: \mu = \mu_0\) vs. \(H_A: \mu \neq \mu_0\). Consider the statistic

\[T = \dfrac{\overline{X} - \mu_0}{s/\sqrt{n}}\]

If \(n\) is sufficiently large, the \(T \approx N(0, 1)\), because \(s \approx \sigma\). However, if \(n\) is small, that approximation will be poor.

However, if the original data are truly normal, then \(T \sim t_{n-1}\). This distribution is exact, regardless of sample size, under the assumption of normality.

Chi-squared goodness-of-fit test

A random variable \(X\) follows a distribution with \(k\) categories if \(X \in \{1,...,k\}\) and the probability that \(X\) is in category \(j\) is \(P(X = j) = p_j\), with each \(p_j \in [0,1]\) and \(\sum \limits_{j=1}^k p_j = 1\). We write \(X \sim Categorical(p_1,...,p_k)\). (This is just a generalization of the Bernoulli to more than two categories).

Suppose that we observe \(X_1,...,X_n \overset{iid}{\sim} Categorical(p_1,...,p_k)\). Let \(n_j = \sum \limits_{i=1}^n \mathbb{I}\{X_i = j\}\) (the number of observations in category \(j\)), and note that \(\sum_j n_j = n\). We are interested in testing the hypotheses \[H_0: (p_1,...,p_k) = (p_{01},...,p_{0k}) \hspace{1cm} H_A: (p_1,...,p_k) \neq (p_{01},...,p_{0k})\] (in other words, are the true probabilities for each category equal to hypothesized probabilities).

The usual statistic used to test these hypotheses is

\[T = \sum \limits_{j=1}^k \dfrac{(n_j - e_j)^2}{e_j}\]

where \(n_j\) is the observed count in category \(j\), and \(e_j\) is the expected count in category \(j\) under \(H_0\). Under \(H_0\), \(T \approx \chi^2_{k-1}\).

Inference with a logistic regression model

Hypothesis testing often involves testing hypotheses like \(H_0: \theta = 0\) for some parameter of interest \(\theta\) (for example, we regularly test whether the coefficients in a regression model are 0). However, there are several critiques of such tests, including that:

  • rejecting the null hypothesis \(H_0: \theta = 0\) does not actually tell us what the value of \(\theta\) is, only that we think it isn’t 0
  • if \(\theta \neq 0\), that still doesn’t make \(\theta\) meaningful. For example, it could be that, actually, \(\theta = 0.000000001\), which is non-zero (but perhaps not meaningfully different from zero)

These critiques are valid, but they do not mean that statistical inference is useless. To address the first criticism, we can create a confidence interval that gives us a reasonable range of values for the unknown quantity. To address the second question, we can test null hypotheses such as \(H_0: \theta \leq \delta\) or \(H_0: |\theta| \leq \delta\) for some value \(\delta\).

In this portion of the assignment, you will construct a confidence interval for a probability of interest using a logistic regression model, and then test a hypothesis about the effect of one of the variables in that model.

Conceptual questions

Statistically “significant”?

Traditionally, we use the phrase “statistically significant” to describe a result in which we reject the null hypothesis. E.g., a scientist might say they found a “statistically significant difference” in the effects of two different drugs.

However, this phrase has been criticized because “significant” doesn’t necessarily mean “important”; in earlier stats courses, for example, you probably discussed the difference between “statistical significance” and “practical significance”.

Because the word “significant” is so potentially misleading, Jeff Witmer (from Oberlin) and others advocate the use of “statistically discernible” rather than “statistically significant”. That is, an observed difference in means is “statistically discernible” if I reject the null hypothesis (but it may or may not be of any real importance!).
