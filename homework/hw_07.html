Sandwich variance – Poisson regression

Suppose that we have iid observations \((\mathbf{x}_i, Y_i)\) from some distribution \(g\). We assume that the data come from the Poisson regression model:

\[ \begin{aligned} Y_i | \mathbf{x}_i &\sim Poisson(\mu_i) \\ \log(\mu_i) &= \mathbf{x}_i^T \boldsymbol{\beta} \end{aligned} \] However, our assumption could be wrong! We therefore decide to use the sandwich variance estimator. Let \(\widehat{\boldsymbol{\beta}}\) be our estimator obtained by maximizing the Poisson likelihood (which may or may not be the true likelihood). Then, recall that we have shown that

\[ \sqrt{n}(\widehat{\boldsymbol{\beta}} - \boldsymbol{\beta}^0) \overset{d}{\to} N(\boldsymbol{0}, \mathbf{S}(\boldsymbol{\beta}^0)) \]

where \(\mathbf{S}(\boldsymbol{\beta}^0)\) is the sandwich variance.

Simulation: the effect of model mis-specification

Now let’s use some simulations to explore the behavior of \(\widehat{\boldsymbol{\beta}}\) when the model is mis-specified. For these simulations, suppose that we observe iid data \((\mathbf{x}_i, Y_i)\) where

\[ \begin{aligned} Y_i | \mathbf{x}_i &\sim NB(\mu_i, \theta) \\ \log(\mu_i) &= \mathbf{x}_i^T \boldsymbol{\beta} \end{aligned} \]

where \(NB(\mu_i, \theta)\) is the negative binomial distribution with mean \(\mathbb{E}[Y_i|\mathbf{x}_i] = \mu_i\) and variance \(Var(Y_i | \mathbf{x}_i) = \mu_i + \frac{\mu_i^2}{\theta}\).

In this model, we can see that we have correctly specified the systematic component of the model (the log mean is a linear function of the explanatory variables). However, we have incorrectly specified the random component. As we saw in HW 1, there is a close connection between the negative binomial and the Poisson. For large \(\theta\), \(NB(\mu_i, \theta) \approx Poisson(\mu_i)\). But for small \(\theta\), the negative binomial has much more variability than the Poisson. So, for small \(\theta\), our Poisson model will substantially underestimate the true variability in the data.

The code below simulates \(n = 200\) observations from the model

\[X_i \overset{iid}{\sim} N(0, 1) \hspace{1cm} Y_i|X_i \sim NB(\mu_i = \exp\{1 + 0.5X_i \}, \theta = 0.5)\] We then fit a Poisson regression model to estimate the coefficients \(\beta_0 = 1\) and \(\beta_1 = 0.5\). Again note that, by construction, our systematic component of the model is correctly specified (the mean function), but our random component (the Poisson distribution) is not correct.

n <- 200
theta <- 0.5
x <- rnorm(n)
mu <- exp(1 + 0.5*x)
y <- rnbinom(n, size = theta, mu = mu)

m1 <- glm(y ~ x, family = poisson)

Comparison with the MLE

Though the Poisson model was mis-specified, \(\widehat{\boldsymbol{\beta}}_{\text{Pois}}\) is still consistent for the true mean function because the systematic component was correct. Furthermore, using the sandwich variance allowed us to correctly estimate \(Var(\widehat{\boldsymbol{\beta}}_{\text{Pois}})\).

The next question to ask is: how does the Poisson estimator compare to the true MLE (i.e., the MLE obtained by correctly assuming a Negative Binomial)?

The estimator \(\widehat{\boldsymbol{\beta}}_{\text{NB}}\) obtained by maximum likelihood estimation with the Negative Binomial distribution can be calculated in R using the glm.nb function from the MASS package. Here is an example:

Moral

The point here is that, if parts of our model are correct (or close enough to correct), then maximum likelihood estimation with a mis-specified model can still be useful for parameter estimation. Furthermore, the sandwich variance can approximate the true variance of our parameter estimates even though the model is mis-specified.

However, we also know that maximum likelihood estimation is asymptotically efficient – maximum likelihood estimation with a correctly-specified model will beat incorrect model + sandwich variance.

Empirical cdfs, quantiles, and M-estimation

So far, our class has focused mostly on parametric estimation: we write down a parametric model for the data, and fit the model using techniques like maximum likelihood estimation. However, estimation need not be parametric – there are a wide variety of nonparametric approaches to estimating distributions and statistical functionals. Here we will explore estimation of the cdf and quantiles for an arbitrary distribution, without requiring any parametric assumptions on the distribution.

Empirical cdf

Let \(Y_1,...,Y_n\) be an iid sample from some distribution with cdf \(F\). No assumptions are made about the form of \(F\). The natural estimator of \(F\) is called the empirical cdf, \(F_n\), and is defined as:

\[F_n(t) = \frac{1}{n} \sum \limits_{i=1}^n \mathbb{I}\{ Y_i \leq t \}\]

Quantiles and M-estimation

For \(p \in (0, 1)\), the \(p\)th quantile of \(F\) is the value \(\theta = F^{-1}(p)\). That is, \(F(\theta) = p\). For example, the median is the quantile with \(p = 0.5\).

We estimate quantiles of \(F\) with the quantiles of the empirical cdf \(F_n\). In particular, the sample \(p\)th quantile is \(\widehat{\theta} = F_n^{-1}(p)\). Written another way, \(F_n(\widehat{\theta}) \approx p\). And to be a bit more formal: \[|F_n(\widehat{\theta}) - p| \leq \frac{1}{n}\] Now, we can write this like an M-estimator! The sample quantile \(\widehat{\theta}\) satisfies

\[ \sum \limits_{i=1}^n \psi(Y_i, \widehat{\theta}) = c_n \tag{1}\] where \[\psi(Y_i, \widehat{\theta}) = \mathbb{I}\{Y_i \leq \widehat{\theta}\} - p\] and \[|c_n| = n|F_n(\widehat{\theta}) - p| \leq 1\]

Equation 1 looks very similar to our definition of an M-estimator. However, we have two small problems to deal with before we can apply M-estimator results. First, the right-hand side of Equation 1 is not 0; it is \(c_n\). Second, the function \(\psi\) has a jump discontinuity at \(Y_i\), which causes problems for differentiation. Fortunately, both of these issues can be accounted for in our theory of M-estimation, as described in the following small extension of the results from class.

Extension: Suppose \(Y_1,...,Y_n\) are iid from some distribution \(F\), and \(\widehat{\boldsymbol{\theta}}\) is an estimator which satisfies \[\sum \limits_{i=1}^n \psi(Y_i, \widehat{\boldsymbol{\theta}}) = \boldsymbol{c}_n\] where \(\boldsymbol{c}_n/\sqrt{n} \overset{p}{\to} \boldsymbol{0}\) as \(n \to \infty\). Furthermore, suppose that \(\psi\) is not necessarily continuous. Then, \[\sqrt{n}(\widehat{\boldsymbol{\theta}} - \boldsymbol{\theta}_0) \overset{d}{\to} N(\boldsymbol{0}, \mathbf{S}(\boldsymbol{\theta}_0))\] where \[\mathbf{S}(\boldsymbol{\theta}_0) = \mathbf{A}^{-1} \mathbf{B} \mathbf{A}^{-1}\] and \[\mathbf{A} = -\frac{\partial}{\partial \boldsymbol{\theta}} \mathbb{E}_F[\psi(Y_i, \boldsymbol{\theta})] \ \Biggr\lvert_{\boldsymbol{\theta} = \boldsymbol{\theta}_0}\] \[\mathbf{B} = \mathbb{E}_F[\psi(Y_i, \boldsymbol{\theta}_0)\psi(Y_i, \boldsymbol{\theta}_0)^T]\] That is, our matrix \(\mathbf{B}\) stays the same as before, but our matrix \(\mathbf{A}\) is slightly different: we have switched the order of integration and differentiation. In other words, \(\mathbf{A}\) is calculated by taking the derivative of \(\mathbb{E}_F[\psi(Y_i, \boldsymbol{\theta})]\) with respect to \(\boldsymbol{\theta}\), and evaluating the derivative at \(\boldsymbol{\theta}_0\).
